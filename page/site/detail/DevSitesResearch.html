개발자 자료용 사이트


• Stack Over Flow

- https://stackoverflow.com/

- 개발자들이 프로그래밍을 하다 막히거나 할 때, 프로그래밍에 대한 질문을 하고 답변을 받는 사이트입니다.


• SourceForge

- https://sourceforge.net/

- 소프트웨어 개발자들을 위해 열려있는, 오픈 소스 소프트웨어 개발관리를 위한 사이트입니다.


• Code Project

- https://www.codeproject.com/

- 소프트웨어 개발자들을 위해 오픈소스와 튜토리얼을 제공하는 사이트입니다.
