노트북

[고려해야할 세부사항]

CPU

- 성능 : i3 < i5 < i7

- 프로그래밍을 구동하는 데 있어 CPU는 중요한 역할을 하기 때문에 i5이상 권장

- 세대 : 10세대가 가장 최신, 금액대가 높으므로 7~8세대 권장

RAM(데이터전송)

- 종류 : 4GB, 8GB, 16GB 등등

- 16GB – 그래픽작업, 8GB - 가상 머신 등 다양한 종류의 개발, 4GB - 평범한 개발

- 4GB와 8GB는 가격차이가 심하지 않음

SDD, HDD(저장공간)

- HDD보다 SDD의 속도가 더 빠름

- 용량은 보편적으로 256GB 또는 512GB를 사용

- 코딩용 노트북이라면 256GB만으로도 충분

그래픽카드

- 프로그래밍만 한다면 내장용 그래픽카드로도 충분

- 간단한 영상편집이나 그래픽 작업도 할 것이면 MX250가 탑재된 노트북 권장

크기, 무게

- 14~16인치 추천

- 무게가 상관없으면 큰 것 추천

운영체제 유무

- 운영체제가 설치 유무로 대략 20만원 차이

- 운영체제 포함 제품 권장
